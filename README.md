# RoadApp

Aplicación Android que desarrollé como proyecto final, donde se pueden generar rutas y dejar opiniones en distintos puntos del mapa.

---

## ¿Qué hace la app?

- Permite buscar o seleccionar un destino en el mapa  
- Genera una ruta desde tu ubicación actual  
- Se pueden agregar opiniones en ubicaciones específicas  
- Cada opinión tiene comentario y puntuación  
- Muestra en el mapa los puntos donde hay opiniones  
- Guarda los datos para que no se pierdan al cerrar la app  

---

## Tecnologías que usé

- Kotlin  
- Android Studio  
- Google Maps API  
- Places API  
- Retrofit  
- Gson  
- SharedPreferences  

---

## Login

El login es simple:  

- Se puede ingresar con cualquier nombre de usuario  
- La contraseña es: **1234**

El usuario se guarda y después se usa para mostrar quién hizo cada opinión.

---

## Guardado de datos

Para guardar la información usé **SharedPreferences**.

Con eso logro que:
- El usuario quede guardado  
- Las opiniones no se pierdan  
- Cuando abrís la app de nuevo, todo sigue ahí
